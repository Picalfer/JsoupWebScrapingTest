# Jsoup Web Scraping Test

[![Kotlin](https://img.shields.io/badge/Kotlin-1.8+-blue.svg?logo=kotlin)](https://kotlinlang.org)
[![Platform](https://img.shields.io/badge/Platform-Android-green.svg?logo=android)](https://www.android.com/)
[![Jsoup](https://img.shields.io/badge/Library-Jsoup-green.svg)](https://jsoup.org/)

Тестовое приложение для изучения веб-скрапинга с помощью Jsoup в Android. Демонстрирует парсинг веб-страниц и получение данных.

## Функциональность

- Парсинг заголовка YouTube страницы
- Асинхронная загрузка веб-контента
- Круглое изображение профиля
- Работа с сетью в фоновом потоке

## Технологии

- Kotlin
- Jsoup
- Coroutines (GlobalScope)
- CircleImageView
- ConstraintLayout
